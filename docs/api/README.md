# hyperliquidx

A reactive wrapper for the Hyperliquid API

## Installation

```bash
npm install hyperliquidx
```

## Usage

```typescript
import { greet } from 'hyperliquidx';

console.log(greet('World')); // Hello, World!
```

## Development

```bash
# Install dependencies
bun install

# Build
bun run build

# Develop with watch mode
bun run dev

# Run tests
bun run test
```
